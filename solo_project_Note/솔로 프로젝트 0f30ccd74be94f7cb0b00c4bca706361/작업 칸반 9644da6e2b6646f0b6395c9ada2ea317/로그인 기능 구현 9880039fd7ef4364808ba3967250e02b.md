# Task 개요

- 로그인 기능을 추가해 줍니다.
    - API 이용한 구글 로그인
- 로그인 한 유저에 대해서만 서비스를 이용할 수 있게 해줍니다.
- 로그인 할 경우 메인페이지로 이동시켜줍니다.
- 서버측에 액세스 토큰을 전달해줍니다.
    
    ```jsx
    const userInfo = await GoogleSignin.signIn();
    const { idToken } = userInfo;
    
    const response = await axios.post("/auth/google", {
      token: idToken,
    });
    ```
